# DigiQuarium

**Blockchain-powered Interactive Aquarium on Ethereum**

DigiQuarium is an educational blockchain platform where users can create unique **fish tokens** on the Ethereum network. The platform integrates real-time blockchain transactions with a fun, interactive aquarium experience.

---

## 🚀 Features
- **Mint Your Own Fish Token**: Powered by Ethereum smart contracts.
- **Full-Stack Blockchain Application**: Real-time transaction visualization.
- **Secure & Transparent**: Powered by Web3 and Metamask.
- **Blockchain Education Tool**: Learn Ethereum through interactive gameplay.

---

## 🛠 Tech Stack
- **Languages**: JavaScript, HTML, CSS
- **Blockchain**: Solidity, Ethereum, Truffle
- **Libraries**: Web3.js
- **Tools**: Ganache, Infura, Metamask
